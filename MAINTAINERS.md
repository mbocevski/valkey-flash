## Contact

- General maintainer questions, patches, review requests — open a GitHub PR or issue
- Security vulnerabilities — [GitHub Security Advisories](https://github.com/mbocevski/valkey-flash/security/advisories/new), see [SECURITY.md](SECURITY.md)
- Code of Conduct enforcement — see [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)

## Becoming a maintainer

Contributors who demonstrate sustained, high-quality engagement — substantial code review participation, consistent contributions across subsystems, and good judgment on scope and risk — may be invited by existing maintainers. No formal application process.
