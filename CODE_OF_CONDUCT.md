# Code of Conduct

This project adopts the Contributor Covenant, version 2.1.

See: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Enforcement

Report Code of Conduct violations privately to the project maintainer at `marko.bocevski@keitaro.com`. Reports are handled confidentially. For violations that also involve a security vulnerability, use [GitHub Security Advisories](https://github.com/mbocevski/valkey-flash/security/advisories/new) instead — see [SECURITY.md](SECURITY.md).

Maintainers will acknowledge reports within 7 days and follow the Contributor Covenant's enforcement guidelines (linked above).
